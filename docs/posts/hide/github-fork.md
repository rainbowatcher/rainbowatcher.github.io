## 建立分支库

1. 在目标仓库的页面上点击 Fork 按钮
2. clone 你的分支仓库到本地
3. 建立相应的特性分支(推荐你这样做)
4. 编写代码
5. pull上游仓库代码，push 到你的分支
6. 去你项目的主页新建 [Pull Request](https://docs.github.com/cn/pull-requests/)

现在你在 Github 上提交了 Pull Request
